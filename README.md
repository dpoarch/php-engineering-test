# PHP-Engineer-test

## Project setup
```
Run Build.sh
```
If Build.sh doesn't failed to start run:
```
docker build -t dockervue .
```

```
docker run -p 8000:80 -it --name dockervue dockervue
```

```
docker-compose up
```

### Project runs at port:
```
http://localhost:8000/
```
